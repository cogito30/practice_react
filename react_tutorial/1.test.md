# React: UI 표현히가

## 목표
- React는 사용자 UI를 렌더랑하기 위한 JS 라이브러리
- UI는 버튼, 텍스트, 이미지와 같은 요소로 구성
- React를 통해 작은 요소들을 재사용 가능하고 중첩할 수 있는 컴포넌트로 조합
